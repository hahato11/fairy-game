# 🦋 Fairy Dress-Up Game | 梦幻精灵换装游戏

A beautiful and interactive dress-up game featuring fairy characters with multiple outfits and hairstyles.

## ✨ Features

- 🎵 **Background Music** - Magical fairy music
- 👗 **30+ Combinations** - 5 hairstyles × 6 dresses
- ⭐ **Rating System** - Real-time outfit scoring
- 📸 **Save Image** - Download your creation as PNG
- 🔗 **Share** - Generate shareable links
- 💾 **Auto Save** - Resume your last outfit
- ✨ **Visual Effects** - Falling petals, twinkling stars, magic circles

## 🚀 Quick Start

Simply open `index.html` in a web browser or deploy to Vercel for online access.

## 📦 Project Structure

```
fairy-game-deploy/
├── index.html          # Main game file
├── 人物底图/           # Character base images
├── 发型/              # Hairstyle images
├── 裙子/              # Dress images
└── README.md          # This file
```

## 🌐 Deploy to Vercel

1. Visit [vercel.com](https://vercel.com)
2. Sign up / Log in with GitHub or Google
3. Click "Add New Project"
4. Drag & drop this folder or import from local
5. Click "Deploy"
6. Share your game URL with friends!

## 🎮 How to Play

1. Click "Start Game" button
2. Choose hairstyles and dresses from the panels
3. See your outfit rating in real-time
4. Save your creation or share with friends
5. Try random combinations for surprises!
